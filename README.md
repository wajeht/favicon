# 🌐 Favicon

automagically grab the favicon of a url


# 📖 Usage

add this to your html:

```html
<img src="https://favicon.jaw.dev?url=<url>" />
```

> [!NOTE]
> the first request will be slow, but the subsequent requests will be cached.
